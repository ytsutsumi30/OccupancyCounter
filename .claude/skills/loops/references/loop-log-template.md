# ループ実行ログテンプレート

scratchpad に `loop-log.md` として作成し、イテレーションごとに追記する。
目的は「次の試行で同じ失敗を繰り返さない」こと。

```markdown
# Loop Log: <タスク名>

- ゴール: <機械判定可能なゴール>
- 判定コマンド: <コマンド>
- 最大試行回数: <N>

## Iteration 1 — <日時>

- 変更点: <何をどう変えたか（ファイル・要点）>
- 検証結果: <判定コマンドの実際の結果。失敗ならエラーの要点>
- 判定: 達成 / 未達 / エスカレーション
- 次の方針: <未達の場合、次に試すこと。前回と同じアプローチでないことを確認>

## Iteration 2 — <日時>
...
```
